객체        메소드 종류
doucument : 웹페이지에 있는 어떤 태그를 삭제하고 싶거나 자식 태그를 추가하고 싶을 때
DOM       : 위 내용을 doucument 객체에서 찾을 수 없을 때
window    : 웹페이지가 아닌 웹브라우저 자체를 제어해야 할 때
  ex. 현재 열려 있는 웹페이지의 주소 알아내기
      새 창 열기
      웹브라우저의 화면 크기를 js로 알아야 할 때

ajax      : 웹페이지를 리로드하지 않고 정보를 변경할 때
cookie    : 웹페이지가 리로드되어도 현재 상태를 유지해야 할 때 -> 사용자를 위한 개인화된 서비스
offline web application: 인터넷이 끊겨도 동작하는 웹페이지를 만들 때
